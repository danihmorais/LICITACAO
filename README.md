# Gerador de Artefatos de Licitação (Lei nº 14.133/2021)

Aplicativo desktop desenvolvido em Python para automatizar, padronizar e otimizar a elaboração dos artefatos da fase interna das contratações públicas. O sistema utiliza inteligência artificial para auxiliar na geração automática do Documento de Formalização de Demanda (DFD), Estudo Técnico Preliminar (ETP) e Termo de Referência (TR), mantendo a identidade visual, estrutura e formatações institucionais do órgão público.

## Visão Geral

A aplicação realiza o preenchimento automatizado de informações fixas da contratação — como Secretaria requisitante, gestor, fiscal e objeto — e utiliza IA para redigir os conteúdos técnicos, jurídicos e mercadológicos dos documentos licitatórios.

O sistema também adapta automaticamente cláusulas relacionadas à vigência, prorrogação e regime de contratação conforme a modalidade selecionada, incluindo:

* Ata de Registro de Preços (ARP);
* Contrato administrativo;
* Contratação direta.

Inclui um sistema de atualização automática integrado com o GitHub Releases, assegurando a execução contínua da versão mais recente disponível.

## Pré-requisitos (Modo Desenvolvedor)

O ambiente de desenvolvimento requer o Python instalado e as seguintes bibliotecas:
